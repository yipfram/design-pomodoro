# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-04

### Added
- Initial release of Zen Pomodoro Timer
- Pomodoro timer with customizable focus and break durations
- Task management system with target and completed pomodoro tracking
- Local storage persistence for tasks
- Desktop notifications for timer completion
- Progressive Web App (PWA) support
- Responsive design for mobile and desktop
- Version display in app footer
- Automated release creation via GitHub Actions

### Features
- 25-minute focus sessions (Pomodoro technique)
- 5-minute short breaks
- Task list with progress tracking
- Clean, minimalist Zen-inspired design
- Browser notifications
- Offline support

## [1.0.1] - 2025-11-05

### Added
- claude.md file with AI version management instructions
- Automated version incrementing guidelines for all future changes
